The Theme Switcher plugin allows you to set dynamically theme to website.

== Description ==

The Theme Switcher plugin allows you to set dynamically theme to website from URL by query string parameter (wptheme) i.e. https://yourwebsite.com/?wptheme=themename

== Installation ==

This section describes how to install the plugin and get it working.

1. Upload the 'theme-switcher' folder to the to the '/wp-content/plugins/' directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Open your website URL with wptheme query string paramater the name of theme you want to load dynamically i.e. https://yourwebsite.com/?wptheme=themename
5. Enjoy!
